# Household Chores Management Web Application

## Overview
This web application helps families manage their household chores effectively. It allows users to assign, track, and manage various household duties in a collaborative manner.

## Features
- **User Authentication**: Secure login and signup functionality.
- **Chore Assignment**: Easily assign chores to family members.
- **Progress Tracking**: View the progress of chores and notify users of upcoming tasks.
- **Notifications**: Get reminders about pending chores through email notifications.
- **User Roles**: Different access levels for family members and administrators.

## How to Use
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/BlurryF4ceOne/App.git
   ```
2. Navigate to the project directory:
   ```bash
   cd App
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
5. Open your browser and navigate to `http://localhost:3000` to access the application.

## Running Locally
To run the application locally:
- Ensure you have [Node.js](https://nodejs.org/) installed.
- Follow the usage instructions listed above.
